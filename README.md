# ECDFC-Story

**Early Childhood DFC Story** — 精進師資素質與特色發展計畫：培育未來幼兒的教育人才

## 計畫簡介

本計畫由國立臺北教育大學執行，聚焦於幼兒教育師資培育，結合 Design for Change（DFC）設計思考與聯合國永續發展目標（SDGs），推動社會情緒學習（SEL）融入幼兒園教學實踐。

## 網站內容

- **首頁** — 計畫願景與重點數據
- **計畫介紹** — 計畫目標、團隊成員
- **成果展示** — DFC-SDGs 課程開發、SEL 教師培訓等執行成果
- **最新消息** — 活動資訊、公告、成果發表
